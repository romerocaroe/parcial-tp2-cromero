# parcial-tp2-cromero

## Setup del proyecto
Parcial de la materia Taller de Programación 2 de la ORT.
### Configuración 
```
npm install
```
```
Configurar las variables de entorno según el contenido del .env.example (pegar en el .env)
```
### Para levantarlo
```
npm start
```
